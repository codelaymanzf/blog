
# Linux基本命令

|  | CentOS | Ubuntu | 备注 |
| :------: | :-------- | :--------| :---|
| 安装软件 | `rpm -i jdk-XXX_linux-x64_bin.rpm` | `dpkg -i jdk-XXX_linux-x64_bin.deb`| |
| 查看软件 | `rpm -qa`| `dpkg -l` | `dpkg -l | grep jdk` |
| 删除软件 | `rpm -e` | `dpkg -r` | |
| `软件管家` | `yum install ` | ` apt-get install` | |
